# 제작 이유
- 글을 매번 써야지 써야지 하다가 정작 실천으로 옮기지 못 하였다.
- 그래서 이렇게 작성 FORM이라도 만들어 놓으면 어느정도 강제성이 생기지 않을까 싶어서 만들었다.
- 초반부터 긴 글을 쓰면 흥미가 떨어지지 않을까 싶어서 1000자 이하로, 기승전결 구조로 만들어보자 생각했다.

# history
- 처음에는 PC 환경에서만 작성하면 되지 않을까 싶었다.
- 하지만 종종 출퇴근 시간, 이동 시간을 무의미하게 보내는 적이 있기에 이 시간에 핸드폰을 이용해서 작성해도 좋을 것 같아 Github Pages를 이용했다.
- PC에서는 txt 파일로 저장이 되지만 모비일에서는 txt 파일을 저장해 봤자 확인이 어려울 것 같아서 메일 발송을 선택했다.(EmailJS 선택 이유)
